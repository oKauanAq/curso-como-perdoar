# Curso Como Perdoar

Landing page estática, responsiva e elegante para divulgação e inscrição do **Curso Como Perdoar**, promovido pelo **Shalom Parnaíba**.

## Objetivo

A página apresenta o curso de forma acolhedora, clara e espiritual, conduzindo a pessoa interessada para a inscrição pelo WhatsApp, sem tom de venda agressiva.

## Informações do curso

- **Nome:** Curso Como Perdoar
- **Tema:** Cura dos relacionamentos através do perdão
- **Período:** 24/06 a 15/07
- **Frequência:** Todas as quartas-feiras
- **Horário:** 19h00
- **Local:** Shalom Parnaíba
- **Endereço:** Av. São Sebastião, 1151
- **Inscrição:** <https://w.app/j8uwfa>

## Arquivos

- `index.html` — estrutura semântica da landing page.
- `style.css` — estilos mobile-first, responsivos e sem dependências externas.
- `script.js` — interação acessível do FAQ.
- `assets/images/` — pasta reservada para imagens do projeto.

## Publicação no GitHub Pages

Este projeto está pronto para GitHub Pages. Para publicar:

1. Envie os arquivos para um repositório no GitHub.
2. Acesse **Settings > Pages**.
3. Selecione a branch principal e a pasta raiz (`/`).
4. Salve e aguarde a geração do link público.
